# 🌤 Weather Dashboard
A real-time weather dashboard built with vanilla JavaScript.
Fetches live weather data using the Open-Meteo API.
## Features
- Search any city in the world
- Displays temperature, wind speed, and weather condition
- Loading state and error handling
- Auto-loads Johannesburg on startup
## Tech Used
- HTML5 & CSS3
- Vanilla JavaScript (ES6+)
- Async/Await
- Fetch API
- Open-Meteo API (free, no key needed)
## What I Learned
- Synchronous vs asynchronous JavaScript
- Promises and async/await syntax
- Fetching data from external APIs
- Handling loading states and errors
- Chaining two API calls in sequence
